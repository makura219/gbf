Help With Notable Features
  • Join Button: If you have the Viramate Chrome extension, have GBF running in another browser tab/window, have enabled the Web API setting in Viramate, and have reloaded the GBF page, the Join button will send you right to the support summon selection page for that raid provided that you meet the requirements for the raid and have enough BP.
  • Click to Copy: Click on any part of a raid's row besides the Join button to copy the raid's ID for easy copying.
  • Desktop Notifications: Click on a desktop notification to copy it's id or to join the raid if you have the Viramate integration setup.
Vertical Layout Options
  • Single Row:The "Single Row" option will just keep adding new selected raids to a single row which means that you will have to scroll horizontally after a certain number.
  • Stacking:The "Stacking" option will keep everything on the same page by putting new selected raids below the first couple like a grid. PLEASE NOTE: The "Stacking" option does not work in Chrome because Chrome has a bug that they haven't fixed yet. I will update the feature when they do fix it.

News - 7/04/2017

WOW! Thanks for every one who donated! During the weekend, I hosted GBFRaiders on a brand spanking new server that won't go down if there is too many people going to it! (Hopefully...)

This update finally has the vertical layout like aikats that people asked for, along with individual notification settings for each raid in that mode. The global settings will override any individual settings whenever they are changed. For example: if you set the sound volume to max in an individual raid's settings, but then change the global volume settings to 50%, the individual setting will need to be changed again. So basically, set the global settings first, then the individual. Also, the individual settings are not saved yet when you refresh the page, they will need to be set again.

To explain the Vertical layout option next to the layout switcher: if you are in the vertical mode, "Single Row" option will just keep adding new selected raids to a single row which means that you will have to scroll horizontally after a certain number. The "Stacking" option will keep everything on the same page by putting new selected raids below the first couple like a grid. PLEASE NOTE: The stacking option does not work in Chrome because Chrome has a bug that they haven't fixed yet. I will update the feature when they do fix it.

I also added alerts for when you try to join a raid when: you have not enough BP to join, the raid is full, the raid is over, and when you are already in the raid. The desktop notification title now changes from the English name to the Japanese name depending on the language that the original tweet was in. Just for fun, I made the join button change colors based on the raid status: blue for unclicked, grey for clicked, red for an error, green for success.
